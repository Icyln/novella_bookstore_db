# Novella Bookstore Database Project

## Project Overview
This project contains the database design and implementation for the **Novella Bookstore System**. It includes the SQL schema, ERD, and full documentation.

## File Structure
- `novella_bookstore.sql`: Main SQL script for database creation and test data.
- `novella_erd.erdplus`: Entity Relationship Diagram file.
- `novella_relational_schema.erdplus`: Relational schema mapping.
- `Novella_Data_Dictionary.xlsx`: Detailed attribute descriptions.
- `novella_bookstore.documentation.pdf`: Final project report.

## How to Use
1. Clone the repository.
2. Run `novella_bookstore.sql` in MySQL Workbench or any SQL client.
3. Review the documentation for business logic and ERD details.
